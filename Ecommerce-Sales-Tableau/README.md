## ✨ Features
- **Dynamic KPI Selector**: Toggle between Revenue, Orders, Profit, Quantity Sold, and Average Order Value using a parameter-driven calculated field.
- **Sales by Category**: Horizontal bar chart with formatted labels showing performance per category.
- **Time Series Trends**: Orders over Time and Sales over Time line charts.
- **Executive KPIs**: Total Sales, YTD Sales, and Adjusted Target Sales displayed in clean cards.
- **Dark theme** with professional, high-contrast design optimized for clarity.

## 🛠️ Tools & Skills Demonstrated
- Tableau Desktop & Tableau Public
- Parameters & Dynamic Calculated Fields
- LOD Expressions (for most recent year totals)
- Formatted Labels & Custom Number Formatting
- Dashboard Layout with Tiled Containers
- Data Visualization Best Practices

## 📈 Key Insights
- Clothing and Accessories consistently rank as top-performing categories by revenue.
- Dynamic switching allows quick comparison across different metrics without rebuilding views.
- Clear visualization of category contribution to overall sales performance.
